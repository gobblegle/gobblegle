How to publish a new board
  1. Fill in the letters above.
  2. Click Download board-config.json.
  3. Replace the board-config.json file in your GitHub repo with the downloaded one.
  4. Commit and push — the live game updates automatically.

Players' progress resets automatically whenever the board changes.
